#  Student Performance Predictor

# Overview :

This project predicts student marks based on study hours, sleep, and attendance using Machine Learning.

# Technologies Used :

* Python
* Pandas
* Scikit-learn

#  Features :

* Predicts marks based on input
* Uses Linear Regression model
* Simple and beginner-friendly

# How to Run :

# Example Input :

* Study Hours: 5
* Sleep Hours: 7
* Attendance: 80

# Output :

Predicted Marks: 65 (approx)

# Model Used :

Linear Regression



* Use larger dataset
* Improve accuracy with advanced models
